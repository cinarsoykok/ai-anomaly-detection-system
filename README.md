# AI-Powered Ensemble Anomaly Detection System

An advanced anomaly detection platform capable of detecting anomalies in both numerical and textual datasets using machine learning and deep learning techniques.

## Features

### Numerical Data Analysis

- Autoencoder Neural Network
- Isolation Forest
- Local Outlier Factor (LOF)
- One-Class SVM
- KNN-based Detection
- Dynamic Ensemble Scoring

### Text-Based Anomaly Detection

- Sentence-BERT Embeddings
- DBSCAN Clustering
- Automatic DBSCAN Parameter Optimization
- Text Cleaning & Augmentation

### Explainable AI

- Feature-level anomaly explanations
- Model contribution analysis
- Anomaly scoring visualization

### User Interface

- Interactive Gradio Interface
- CSV Upload
- Result Export
- Anomaly Preview Table

## Technologies

- Python
- TensorFlow
- Scikit-Learn
- Sentence-BERT
- DBSCAN
- Gradio
- Pandas
- NumPy
